Privacy Policy

Last updated: December 2024

Summary

This extension processes web page content locally on your device to detect AI-generated text. We do not collect, store, or transmit your browsing data.

Data Collection

We do not collect:

  • Browsing history
  • Page content
  • Personal information
  • Usage analytics

Data Storage

The extension stores the following locally on your device only using Chrome's storage API:

  • Your whitelist/blacklist preferences
  • User corrections (marked as human/AI)
  • Extension settings

This data syncs across your Chrome browsers if you're signed into Chrome, via Google's infrastructureâ€”not ours.

Data Processing

All content analysis happens locally in your browser. No page content is sent to external servers.

Premium Features (if applicable)

If you opt into premium features requiring AI verification:

  • Only ambiguous text snippets are sent to our API
  • Snippets are processed and immediately discarded
  • No snippets are stored or used for training

Third Parties

We do not share any data with third parties.

Permissions Justification

  • activeTab / tabs: To read page content for analysis
  • storage: To save your preferences locally
  • contextMenus: To enable right-click options

Changes

We'll update this policy as needed. Continued use constitutes acceptance.
